# Antigravity用 UI/UX刷新プロンプト

以下をそのまま Antigravity に入力して使ってください。

```text
あなたは教育サービスのCV最適化に強いシニアプロダクトデザイナー兼フロントエンドデザイナーです。
既存サイト（Astro静的）を、見た目と導線を大幅刷新してください。

## 目的
- 最重要CV: 初回無料面談（30分）の予約
- 対象: 中高生本人 + 保護者（特に保護者の不安解消を先行）
- 成果指標: /education への遷移率、LINEクリック率、予約ページクリック率

## ブランド体系（固定）
- ホーム: AIYuMe
- 教育: AIYuMe Learning（主役）
- クリエイター: AIYuMe Studio
- 法人AI: AIYuMe Automation
- ドメイン: https://ai-yu-me.com

## 必須要件
1. サイト構成は維持:
   - /
   - /education
   - /creator
   - /automation
   - /privacy
   - /contact
2. /education を主役化し、他2事業は情報密度を下げる
3. 誇大広告禁止（合格保証・短期で必ず伸びる等NG）
4. v1は静的運用（サーバー処理なし）
5. CTAは「LINE相談」と「予約ページ」の2本
6. モバイルでの可読性・タップ導線最優先

## 期待するデザイン方向
- 第一印象: 信頼感・整理感・安心感（煽らない）
- 情報設計: 「不安の言語化 → 解決手段 → 進め方 → FAQ → CTA」
- ヒーロー: 一目で対象・価値・行動が分かる
- 視認性: 見出し階層、余白、CTAコントラストを明確化
- コンポーネント: 再利用しやすい形で統一
- アクセシビリティ: コントラスト、フォーカス、44px以上のタップ領域

## /education の必須セクション順
1. Hero（価値 + 対象 + CTA）
2. よくある詰まり（3〜5項目）
3. 解決アプローチ（仕組み化 / 個別最適化 / 週次レビュー）
4. 提供内容（2プラン）
5. 進め方（面談→診断→2週間プラン→継続）
6. 保護者の関与度
7. FAQ（10〜14個）
8. CTA再掲（LINE + 予約）

## 実装制約
- Astro + TypeScript + CSS（Tailwind不要）
- 既存の `src/config.ts` の値参照を維持
- 既存の主要コンポーネント構造を活かす:
  - Header
  - Footer
  - CTAButton
  - FAQAccordion
- SEO項目（title/description/OG/canonical）を壊さない

## 出力してほしいもの
1. 新UIのデザインコンセプト（短く）
2. 画面別の改善ポイント（/ /education /creator /automation）
3. 実装差分（ファイル単位）
4. 主要CTA導線の改善理由
5. モバイル時の挙動説明

## コピーのトーン
- 煽りすぎない
- 親の不安を先に受け止める
- 「権威」ではなく「運用の再現性」で信頼を作る

## 禁止事項
- 競合文言のコピー
- 過剰な装飾で可読性を落とすこと
- 教育以外の2事業を前面に出し過ぎること

まずは /education のCV導線から優先して刷新し、その後 / の全体導線を最適化してください。
```
